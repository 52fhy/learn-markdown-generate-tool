$ npm install idoc -g

源码仓库 #

仓库地址【https://github.com/jaywcjlove/idoc】 欢迎提需求和bug，提交地址issues 。

新版本体验 # 


# 1. 先克隆`idoc`到本地
$ git clone https://github.com/jaywcjlove/idoc.git

# 2. 卸载已安装的idoc
$ npm uninstall -g idoc

# 3. 进入下载好的 idoc 目录建立一个软连
$ npm link

# 4. 这个时候就可以使用了(可以改改源码进行玩耍)
$ idoc -h
